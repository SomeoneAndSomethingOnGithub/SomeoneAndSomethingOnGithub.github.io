Question 1

Given a function \(f:\mathbb{R}\to\mathbb{R}\) such that it satisfies \[f(x+y)=f(x)+f(y), \quad xf(x)=f(x^2)\]

Suppose the function can be expressed as \(f(x)=xc+d\), where \(c,d\) are arbitrary constants. Find \(c+d\):

Observe by plugging in \(x=x+1,xf(x+1)+f(x+1)=f(x^2+2x+1)\)

\(\implies xf(x)+x+f(x)+(x+1)f(1)=xf(x)+2f(x)+f(1)\)

\(\implies f(x)+(x+1)f(1)=2f(x)+f(1)\)

\(\implies f(x)=xf(1)=xc\)


Question 2

Given \(a,b,c\in\mathbb{Z}^{+}\) such that it satisfies \[a^2+b+c=abc\]

Find the sum of all possible values of \(a+b+c\):

Observe that \(\implies a(bc-a)=b+c\)

\(\implies \frac{bc}{2} \le b+c\)

The rest of the problem is considered trivial and as such will be left as an exercise to the reader.
